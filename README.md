# Gestion Tareas API

## Descripción

API para la gestión de tareas, permite crear, editar, eliminar y listar tareas.

## Tecnologías

- NodeJS
- NestJS
- MongoDB

## Endpoints

### Tareas

- **GET** /tareas
- **GET** /tareas/:id
- **POST** /tareas
- **PUT** /tareas/:id
- **DELETE** /tareas/:id

## Despliegue

### Render

- [https://gestion-tareas-api.onrender.com](https://gestion-tareas-api.onrender.com)

## Ejecución

### Local

```bash
# Instalar dependencias
$ npm install

# Ejecutar aplicación
$ npm run start

# Ejecutar aplicación en modo desarrollo
$ npm run start:dev

# Ejecutar aplicación en modo producción
$ npm run start:prod
```
